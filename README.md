# فرصة (Fursa) — Landing Page

صفحة هبوط (Home Page) لمنصة فرصة، مبنية بـ Next.js 14 (App Router) + TypeScript + Tailwind CSS.
عربي RTL بشكل أساسي، وقابلة للتحويل للإنجليزية لاحقًا بسهولة.

## التشغيل محليًا

```bash
npm install
npm run dev
```

افتح http://localhost:3000

## هيكلة المشروع

```
app/
  layout.tsx        # الخطوط + RTL + الميتاداتا
  page.tsx           # يجمع كل أقسام الصفحة
  globals.css

components/
  ui/                 # عناصر قابلة لإعادة الاستخدام (Button, Badge, Container, SectionHeading, Reveal)
  layout/             # Navbar, Footer
  sections/           # كل قسم من أقسام الصفحة الرئيسية (Hero, Services, FAQ...)

lib/
  data.ts             # كل نصوص ومحتوى الصفحة في مكان واحد (سهل التعديل بدون لمس التصميم)
```

## لماذا هذه البنية؟

- **`lib/data.ts`** يفصل المحتوى عن التصميم — لتغيير أي نص أو إضافة خدمة جديدة، عدّل هذا الملف فقط.
- **`components/ui`** عناصر أساسية (زر، بادج، عنوان قسم) تُستخدم في كل الصفحة، وستُستخدم لاحقًا في صفحات أخرى من المنصة (تسجيل الدخول، لوحة التحكم...).
- كل قسم في `components/sections` مستقل تمامًا، فيمكن حذف/ترتيب/إعادة استخدام أي قسم بسهولة.

## نظام التصميم (Design Tokens)

معرّف في `tailwind.config.ts` تحت `colors.fursa`:

| اللون | الاستخدام |
|---|---|
| `fursa-navy` `#0F2A43` | نصوص أساسية، خلفية الفوتر والإحصائيات |
| `fursa-blue` `#2E6DA4` | اللون الأساسي للعلامة (أزرار، روابط) |
| `fursa-teal` `#1C9C8E` | لون ثانوي (تعليم/تطوير) |
| `fursa-green` `#4CAF6D` | لون تمييز للفرص/الإنجاز (استخدام محدود) |
| `fursa-mist` `#F6F9FC` | خلفية القطاعات البديلة |

الخطوط: **IBM Plex Sans Arabic** (عربي) + **Inter** (أرقام/لاتيني)، مضبوطة عبر `next/font/google` في `app/layout.tsx`.

## العنصر المميز (Signature Element)

القطعة المدارية (Orbit) في الـ Hero (`components/sections/Hero.tsx`) امتداد تصميمي مباشر لشعار فرصة: نقطة مركزية (المستخدم) تدور حولها فرص التدريب، الوظائف، المسابقات، والتطوع — بنفس فكرة الشعار الأصلي لكن كعنصر تفاعلي حي.

## الخطوة القادمة

هذه الصفحة الرئيسية فقط، بالاتفاق. الخطوات التالية المقترحة:
1. صفحة تسجيل الدخول / إنشاء حساب
2. لوحة تحكم الباحث عن عمل (Profile Page)
3. لوحة تحكم الشركات
4. صفحة تفاصيل الفرصة الواحدة
